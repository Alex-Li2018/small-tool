# small-tool
js小型工具库的总结

### 概述
js小型工具库，基于js语言总结工作中常用的函数,类的一个工具库<font size="28">🔨</font>。

### 目录说明

    |-- src
        |-- ClipboardAction.js 复制功能
        |-- cli.js node的CLI
        |-- file_CLI.js node的文件CLI
        |-- mapUtils.js 地图操作API
        |-- no-console.js 判断文件下是否有console
        |-- simplifyMutiPoints.js 点抽稀算法
        |-- statisticsCodeLine.js 统计代码行
        |-- type.js 类型检测函数
        |-- date.js 格式化时间的库
    |-- record.txt 测试文件
